# Kansong App (간송 앱)

### 🚀 React + Vite 기반 프로젝트  

이 프로젝트는 React와 Vite를 기반으로 제작된 **간송 미술관 웹사이트**입니다.  
다음과 같은 주요 기능을 제공합니다:  

- **공지사항 게시판 (CRUD)**
- **보도자료 게시판 (CRUD)**
- **반응형 UI**
- **Firebase 기반 게시판 및 관리자 로그인 기능**
- **GSAP 및 Quill을 활용한 동적 애니메이션 및 텍스트 편집**

---

## 🛠️ 주요 기술 스택  

- **🖥️ Frontend:** React, Vite, TailwindCSS, GSAP, Quill  
- **⚙️ Backend:** Firebase (Firestore, Authentication)
